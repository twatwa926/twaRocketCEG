# RocketCEG 😡

一个 Minecraft 1.20.1 Forge 模组，实现太空探索功能。😡

## 功能特性 😡

- 🌞 立体日月渲染（使用原版材质）😡
- 🌍 立方体星球系统（地球、月球、火星、金星）😡
- 🚀 轨道维度 😡
- 🌌 无缝维度切换 😡
- 🎮 火箭系统 😡

## 版本信息 😡

- Minecraft: 1.20.1 😡
- Forge: 47.3.0 😡
- Java: 17+ 😡

## 快速开始 😡

### 编译项目 😡

```bash
./gradlew build
```

### 运行客户端 😡

```bash
./gradlew runClient
```
